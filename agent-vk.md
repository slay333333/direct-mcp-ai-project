# VK Реклама — Инструкции агента

## Философия

### Структура VK Ads

Иерархия объектов аналогична Директу:
```
Кампания → Группа объявлений → Объявление
```

- **Кампания** — бюджет, цель продвижения, расписание
- **Группа объявлений** — таргетинг (аудитория, гео, устройства, интересы), ставки
- **Объявление** — креатив (изображение/видео), текст, ссылка

### Отличия от Яндекс Директа

| Параметр | Яндекс Директ | VK Реклама |
|----------|---------------|------------|
| Основной трафик | Поисковые запросы | Соцсети и партнёрская сеть |
| Таргетинг | Ключевые фразы + автотаргетинг | Демография, интересы, ретаргетинг, контекстные фразы |
| Формат объявлений | Текстовые (поиск) | Медийные (изображения, видео, карусели) |
| Модель оплаты | CPC / оптимизация конверсий | CPC / CPM / оптимизация конверсий |
| Аудитории | Поисковый интент | Социальные характеристики + поведение |

### Принципы работы
- **Фильтрация** — как и в Директе, не запрашивай всё подряд. Фильтруй по статусам, используй пагинацию.
- **Экономия токенов** — запрашивай только нужные данные. При большом количестве объектов используй limit и offset.
- **Верификация** — перед созданием/изменением кампаний убедись в правильности параметров. Переспроси при сомнениях.
- **Пошаговость** — сложные операции выполняй поэтапно с подтверждением.

## MCP-инструменты

MCP-сервер: `vk-ads` (эндпоинт `https://lidfly.ru/mcp/vk`)

### Кампании

**Получение и управление:**
- Получение списка кампаний (с фильтрацией по статусу)
- Создание кампании (цель, бюджет, расписание)
- Редактирование кампании (название, бюджет, статус)
- Массовое управление (остановка, запуск, удаление нескольких кампаний)

**Статусы кампаний:**
- Активная — показы идут
- Остановлена — приостановлена рекламодателем
- Архивная — завершена, в архиве

### Группы объявлений

- Создание группы с таргетингом
- Редактирование таргетинга и ставок
- Массовые операции с группами
- Привязка аудиторий ретаргетинга

**Таргетинг в группах:**
- Регион (гео)
- Пол, возраст
- Интересы и поведение
- Устройства (мобильные/десктоп)
- Контекстные фразы
- Аудитории ретаргетинга / look-alike

### Объявления (11 инструментов)

- Создание объявления с креативом
- Редактирование текста и ссылок
- Загрузка изображений
- Модерация
- Шаблоны объявлений
- Массовые операции

**Форматы:**
- Изображения
- Видео
- Карусели
- Универсальные записи

### Контент

- Загрузка изображений для объявлений
- Управление креативами

### Статистика

- Показы, клики, CTR
- Расход, CPC, CPM
- Конверсии, CPA
- Охват и частота
- Разбивка по периодам (день/неделя/месяц)

### Аудитории (20 инструментов)

- **Ремаркетинг** — сегменты по действиям на сайте
- **Списки ретаргетинга** — загрузка клиентских баз
- **Look-alike** — поиск похожих аудиторий
- Управление сегментами
- Массовые операции с аудиториями

### Пакеты размещения

- Форматы рекламы и доступные таргетинги
- Просмотр характеристик пакетов

### Прогнозы

- Оценка охвата при разных ставках
- Прогноз показов и кликов

### Контекстные фразы

- Создание и управление фразами для таргетинга
- Аналог ключевых фраз Директа, но для соцсети

### Лид-формы

- Создание форм сбора лидов
- Управление лид-формами в объявлениях

### Справочники

- Регионы (для гео-таргетинга)
- Устройства
- Категории приложений
- Другие справочные данные

### Агентства

- Управление клиентами агентства
- Трансферы средств между клиентами

## Таргетинг

### Демография и интересы
- **Пол** — мужчины/женщины/все
- **Возраст** — диапазоны (18-24, 25-34, 35-44, 45-54, 55+)
- **Интересы** — категории (авто, красота, финансы, недвижимость и т.д.)
- **Поведение** — действия пользователей в соцсетях

### Контекстные фразы
Аналог ключевых фраз Директа — показ объявлений пользователям, которые искали определённые запросы. Используй для:
- Горячей аудитории (искали продукт/услугу)
- Дополнения к таргетингу по интересам

### Ретаргетинг и look-alike
- **Ретаргетинг** — показ тем, кто уже взаимодействовал с сайтом/приложением
- **Look-alike** — расширение аудитории на основе существующей базы клиентов
- **Списки** — загрузка email/телефонов клиентов для точного таргетинга

### Устройства
- Мобильные (iOS / Android)
- Десктоп
- Все устройства

## Типовые сценарии

### Создание кампании с нуля

1. **Определи цель** — трафик, конверсии, охват, лиды
2. **Настрой кампанию** — название, бюджет (дневной/общий), расписание
3. **Создай группу** — таргетинг (гео, демография, интересы, контекстные фразы)
4. **Подготовь креатив** — загрузи изображение/видео
5. **Создай объявление** — текст, ссылка, креатив, CTA
6. **Отправь на модерацию**
7. **Покажи итоговый отчёт** — ID кампании, группы, объявления, статус модерации

**Перед созданием:**
- Уточни цель и бюджет у пользователя
- Предложи структуру таргетинга
- Согласуй тексты и креативы
- Жди подтверждения перед выполнением

### Анализ эффективности

1. Получи статистику кампаний за период
2. Сравни KPI: CTR, CPC, CPA, конверсии
3. Найди лучшие и худшие группы/объявления
4. Проверь аудитории — какие сегменты конвертируют лучше
5. Предложи оптимизацию:
   - Отключить неэффективные группы/объявления
   - Перераспределить бюджет в пользу конверсионных
   - Расширить look-alike на основе лучших сегментов
   - Обновить креативы (баннерная слепота)

### Масштабирование

1. Определи работающие связки (аудитория + креатив)
2. Создай look-alike аудитории на базе конверсий
3. Расширь гео (новые регионы)
4. Протестируй новые форматы объявлений
5. Увеличь бюджет постепенно (не более 20-30% за раз)

## Связка VK + Директ

При работе с обеими платформами одновременно:
- **Разные аудитории** — Директ ловит поисковый интент, VK работает с социальным поведением
- **Ретаргетинг** — посетителей с Директа можно догонять в VK через ремаркетинг
- **Контекстные фразы VK** — используй те же ключевые фразы что и в Директе для охвата аудитории в соцсетях
- **Сравнение CPA** — сравнивай стоимость конверсии между платформами для оптимального распределения бюджета
